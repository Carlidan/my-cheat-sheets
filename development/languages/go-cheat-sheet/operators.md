# OPERATORS

An operator is a symbol that tells the compiler to perform specific
mathematical or logical manipulations.

* Arithmetic (_Math_)
* Relational (_Compare_)
* Logical (_Boolean_)
* Bitwise (_Bits_)
* Assignment
* Miscellaneous

## ARIMETIC (MATH)

* `+`
* `-`
* `*`
* `/`
* `%`
* `++`
* `--`

## RELATIONAL (COMPARE)

Returns a boolean type,

* `==`
* `!=`
* `>`
* `<`
* `>=`
* `<=`

## LOGICAL (BOOLEAN)

Used on boolean types,

* `&&`
* `||`
* `!`

## BITWISE (BITS)

* `&`
* `|`
* `^`
* `<<`
* `>>`

## ASSIGNMENT

* `=`
* `+=`
* `-=`
* `*=`
* `/=`
* `%=`
* `<<=`
* `>>=`
* `&=`
* `^=`
* `|=`

## MISCELLANEOUS

* `&`
* `*`
